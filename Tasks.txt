1. Установите Python 3.8 с официального сайта
2. Установите модули Python, требуемые для работы приложения. Для этого можно установить их в виртуальную среду или глобально.
2.1 Чтобы установить модули в виртуальную среду, необходимо:
* В консоли с правами администратора прописать: pip install virtualenv
* Перейти в корневую папку проекта в консоли: cd <path to project>
* Ввести команду: virtualenv <virtual enviroment name>
* Активировать виртуальную среду: <virtual enviroment>\scripts\activate
* Ввести команду: pip install -r requirements.txt
Таким образом, мы установили зависимости, необходимые для запуска проекта отдельно от глобальных пакетов. Далее необходимо настроить вашу IDE таким образом, чтобы она по умолчанию использовала в проекте созданную виртуальную среду.
2.2 Если вы не хотите использовать виртуальную среду, можно установить модули глобально:
* Запустить консоль с правами администратора и перейти в корневую папку проекта: cd <path to project>
* В консоли прописать: pip install -r requirements.txt
